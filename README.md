# rocket-sim

Super minimal, 1D rocket launch and landing simulation.

## features:
- [x] takeoff
- [x] hover
  - [x] PID controll
- [x] landing
  - [x] distance-till-zero-velocity calculation
  - [x] smoothened  touchdown
- [ ] smooth (realistic) throttling
- [ ] * drag
- [ ] setting limits
  - [ ] lift off velocity
  - [ ] lift off acceleration
  - [ ] maximum altitude
  - [ ] landing velocity
  - [ ] landing acceleration
- [ ] * real rocket numbers
  - [ ] * New shepard
  - [ ] * starship
  - [ ] * Falcon 9
- [ ] * 2D
  - [ ] * yaw controll
  - [ ] * location targetting
  - [ ] * laiding
